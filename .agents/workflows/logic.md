# Lógica de Negocio - Resumen

1. **Objetivo:** Usuarios adquieren aviones y licencias para volar.
2. **Vuelos:** Creados por `weifly` con recompensas.
3. **NFTs:** Cada vuelo genera un NFT.
4. **Callsign:** Vuelos validados con un callsign asignado al simulador.
5. **Combustible:** Staking de `AIRL` produce `AIRG`.
   - Ratio: **100 AIRG/día** por cada **1 AIRL**.
